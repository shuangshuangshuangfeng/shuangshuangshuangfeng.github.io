VTK-同构坐标

 当我们在三维空间中创建对象，显示的时候是把它投影到二维图像平面上，为了以一种更自然的表示方式，引入了同构坐标（homegeneous coordinates）的概念。对于三维空间中的坐标点(x, y, z)用同构坐标表示就是(Xh, Yh, Zh, Wh),同构坐标和笛卡尔坐标的变化如下：

  • x = Xh/Wh
  • y = Yh/Wh
  • z = Zn/Wh

使用同构坐标表示一个无穷远点时，就可以设置Wh为零，这样变换就是以一种4*4的矩阵形式出现。变换矩阵再计算机图形学中应用是非常广泛的，使用4*4变化矩阵可以实现平移、缩放、旋转等很多操作。

VTK-视口

 视口的含义是在VTK一个窗口中可以显示多个小窗口，而每一个小窗口称作视口(viewport).

‘’’
from vtk import
from PyQt5 import

cone = vtkConeSource()
cube = vtkCubeSource()
cylinder = vtkCylinderSource()
sphere = vtkSphereSource()

coneMapper = vtkPolyDataMapper()
coneMapper.SetInputConnection(cone.GetOutputPort())

cubeMapper = vtkPolyDataMapper()
cubeMapper.SetInputConnection(cube.GetOutputPort())

cylinderMapper = vtkPolyDataMapper()
cylinderMapper.SetInputConnection(cylinder.GetOutputPort())

sphereMapper = vtkPolyDataMapper()
sphereMapper.SetInputConnection(sphere.GetOutputPort())

coneActor = vtkActor()
coneActor.SetMapper(coneMapper)

cubeActor = vtkActor()
cubeActor.SetMapper(cubeMapper)

cylinderActor = vtkActor()
cylinderActor.SetMapper(cylinderMapper)

sphereActor = vtkActor()
sphereActor.SetMapper(sphereMapper)

renderer1 = vtkRenderer()
renderer1.AddActor(coneActor)
renderer1.SetBackground(1.0,0.0,0.0)
renderer1.SetViewport(0.0,0.0,0.7,1.0)

renderer2 = vtkRenderer()
renderer2.AddActor(cubeActor)
renderer2.SetBackground(0.0,1.0,0.0)
renderer2.SetViewport(0.7,0.0,1.0,0.33)

renderer3 = vtkRenderer()
renderer3.AddActor(cylinderActor)
renderer3.SetBackground(0.0,0.0,1.0)
renderer3.SetViewport(0.7,0.33,1.0,0.66)

renderer4 = vtkRenderer()
renderer4.AddActor(sphereActor)
renderer4.SetBackground(1.0,1.0,0.0)
renderer4.SetViewport(0.7,0.67,1.0,1.0)

renWin = vtkRenderWindow()

renWin.AddRenderer(renderer1)
renWin.AddRenderer(renderer2)
renWin.AddRenderer(renderer3)
renWin.AddRenderer(renderer4)
renWin.SetSize(640, 480)
renWin.Render()
renWin.SetWindowName(“ViewPort”)

interactor = vtkRenderWindowInteractor()
interactor.SetRenderWindow(renWin)
renWin.Render()
interactor.Initialize()
interactor.Start()

‘’’

vtkRenderer继承了vtkViewport，所以每一个vtkRenderer就是一个视口

vtkRenderer.SetViewport(x1, y1, x2, y2)函数用于设置视口坐标，(x1, y1)是视口的左下角的坐标，(x2,y2)是视口的右下角坐标。
